  1. Download the VStats JAR here.
  2. Open a Java project in VS Code and click on a .java file. Towards the bottom of the File Explorer window, you should see a dropdown menu called Java Projects.
  3. Click on the Java Projects menu and locate the Referenced Libraries folder.
  4. Hover your cursor over the Referenced Libraries folder and click on the "+" icon. Then add the VStats.jar file that you downloaded.
  5. Done!
